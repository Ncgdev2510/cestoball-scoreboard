Scoreboard Pro

Resumen rápido para instalación y licencia (1 página)

Requisitos: Windows 64 bits. Necesitás el instalador y el archivo license.lic generado para esta PC (no sirve copiarlo de otra máquina).

Instalación

  • Ejecutá el instalador (.exe) y seguí el asistente.
  • Si Windows muestra una advertencia de seguridad, solo continuá si confiás en el origen del programa.

Dónde poner license.lic

Elegí una ubicación:

  • Misma carpeta que Scoreboard Pro.exe (donde quedó instalado).
  • O: %APPDATA%\scoreboard-pro\ (pegá esa ruta en la barra del Explorador de archivos).

Nombre exacto del archivo: license.lic

Primera apertura

  • Si ves Activación requerida: pulsá Copiar junto al ID de hardware, enviámelo y te mandamos el license.lic. Guardalo en una de las carpetas de arriba y pulsá Reintentar.
  • Si abre el panel directamente, la licencia ya está bien.

Segunda pantalla (solo marcador)

Abrí la app con ?view=scoreboard en la dirección, o usá el acceso directo que te indiquemos.

Cambié de PC

Necesitás un nuevo license.lic. Escribinos con el ID que muestra la pantalla de activación.

Para más detalle pedí la guía completa del usuario. Podés guardar esta página como PDF desde el navegador: Archivo → Imprimir → Guardar como PDF.
